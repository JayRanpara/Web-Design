Concept Of WWW

  • It stands for world Wide Web.
  • It is basically a system of internet servers that support specially formatted documents.
  • It is a collection of text pages,vidoe,digital photographs and animation we can access over the internet.
  • User access the World Wide Web facilities via client called a web browser.

HTTP Protocol

  1. HTTP stands for Hypertext Transfer Protocol.
  2. HTTP is based on the client-server architecher model and a stateless request/responce protocol.
  3. 404 File Not Found is a common HTTP status code.
